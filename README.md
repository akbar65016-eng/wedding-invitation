# wedding-invitation
An online wedding invitation website designed primarily for mobile devices.

## Status
Currently under development.

## Sections
- Cover
- Opening
- Introduction
- Bride & Groom
- Countdown
- Event & Location
- Love Story
- Gallery
- Gift
- Wishes & RSVP

## Notes
This website is optimized for mobile viewing. Desktop and tablet users will be prompted to open the invitation on a mobile device for the best experience.
